1.Responsive Website

This Template has a Responsive layout and is based on the Media Querries & Viewport Tags. I have made all the pages responsive by including media querries in all the CSS and viewport tags in all the HTML.
I have made the navigation reposnsive by navigation.js javascript file.
I have made the camera slide show responsive by the jquery.mobile.customized.min.js javascript file.

1.1 Viewport Tag in HTML

The given example demonstrates the use of viewport tag in the Master Page.
The width property controls the size of the viewport. It can be set to a specific number of pixels like
width=600 or to the special value device-width value which is the width of the screen in CSS pixels at a scale of 100%.

(There are corresponding height and device-height values, which may be useful for pages with elements that change size or position based on the viewport height.)
